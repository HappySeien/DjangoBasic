5. (*) В качестве пакета создания виртуального окружения используйте программный комплекс pyenv.

Установил pyenv и создал окружение через него.
 Просмотр установленных версий python - pyenv versions;
 Для создания окружения - pyenv virtualenv версия_интерпритатора имя;
 Активация - pyenv activate имя;
 Деактивация - pyenv deactivate;
 Список окружений - pyenv virtualenvs;
 Установка глобальной версии интерпритатора - pyenv global версия;
 Установка локальной версии - pyenv local версия;
